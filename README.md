# Safaris&Tours Ltd CRM System

A full-stack CRM application for managing safari operations, built with React, Vite, Express, and Firebase.

## Features
- **Role-Based Access Control (RBAC):** Automatic redirection and permission management based on staff email.
- **Lead Management:** Capture and track inquiries from the public website.
- **Customer CRM:** Centralized database for traveler profiles and history.
- **Tour & Booking Management:** Tools for sales agents to manage itineraries and reservations.
- **Marketing & Reports:** Analytics and campaign management for business growth.

## Quick Start
1. **Staff Login:** Navigate to `/admin` and sign in with your authorized Google account.
2. **User Manual:** For detailed instructions on roles and features, see [USER_MANUAL.md](./USER_MANUAL.md).
