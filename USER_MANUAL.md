## 3. Core Modules
### 3.1 Dashboard
The central hub for the Super Admin. It provides a real-time overview of:
- Total Revenue
- Active Bookings
- New Leads
- Customer Satisfaction metrics

### 3.2 Customers
Manage the database of all past and current travelers.
- View customer profiles and contact details.
- Track customer history and preferences.

### 3.3 Leads
Track potential clients who have inquired via the website.
- Review new inquiries.
- Assign leads to sales agents.
- Update lead status (New, Contacted, Qualified, Lost).

### 3.4 Tours
Manage the catalog of safari packages.
- View itineraries for Maasai Mara, Amboseli, Serengeti, etc.
- Check pricing and availability.

### 3.5 Bookings
The operational core for the Sales team.
- Manage confirmed reservations.
- Track payment statuses.
- Handle logistics for upcoming tours.

### 3.6 Complaints (Support)
Dedicated section for resolving guest issues.
- Log and track customer feedback.
- Ensure timely resolution of service complaints.

### 3.7 Marketing
Tools for the Marketing Manager to drive growth.
- Manage promotional campaigns.
- Analyze lead sources and conversion rates.

### 3.8 Reports
Data-driven insights for management.
- Generate sales performance reports.
- Analyze seasonal trends and destination popularity.

## 4. Troubleshooting
### 4.1 Login Issues
- **Popup Blocked:** If the Google sign-in window doesn't appear, check your browser's address bar for a "Popup Blocked" icon and select "Always allow popups for this site."
- **Access Denied:** Ensure you are using one of the authorized emails listed in Section 2.2.
- **Concurrent Requests:** If you see an error about a "pending promise," wait a few seconds and try clicking the sign-in button again.
